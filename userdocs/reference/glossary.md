# Glossary

| Term | Meaning |
|---|---|
| **ATL** | Acute Training Load — 7-day EWMA of daily TSS; "fatigue". |
| **CP** | Critical Power — highest metabolically sustainable power; ≈ FTP. Fitted from best efforts. |
| **CSS** | Critical Swim Speed — swim threshold pace (sec/100 m). |
| **CTL** | Chronic Training Load — 42-day EWMA of daily TSS; "fitness". |
| **Decoupling (Pw:Hr)** | % drift of output-per-heartbeat between ride halves; aerobic endurance marker. |
| **Durability** | Best power after N kJ of accumulated work, vs fresh; fade %. |
| **EA** | Energy Availability — intake minus exercise energy, normalized to body mass; chronic low EA → RED-S risk. |
| **EF** | Efficiency Factor — NP (or speed) per heartbeat; aerobic progress tracker. |
| **EWMA** | Exponentially weighted moving average — the PMC's smoothing. |
| **FTP** | Functional Threshold Power — configured threshold; denominators of IF and power zones. |
| **IF** | Intensity Factor — NP/FTP (bike), threshold/actual pace (run), CSS-relative (swim). |
| **kJ tier** | Accumulated-work bucket (500/1000/1500/2000 kJ) used by durability. |
| **LTHR** | Lactate Threshold Heart Rate. |
| **Macrocycle** | The season: a dated block anchored to an A-race, containing phases with weekly TSS targets. |
| **NP** | Normalized Power — 4th-power-weighted average; what the ride felt like. |
| **PMC** | Performance Management Chart — CTL/ATL/TSB over time. |
| **Polarized / pyramidal / threshold** | Intensity-distribution patterns classified from low/moderate/high band shares. |
| **Quadrant analysis** | Pedal force vs pedal velocity shares (AEPF/CPV) around CP at a reference cadence. |
| **Ramp rate** | CTL change per 7 days; weekly alert threshold at +8. |
| **rTSS / sTSS** | Run / swim TSS variants (pace-based; swim uses IF³). |
| **TSB** | Training Stress Balance — yesterday's CTL − ATL; "form". |
| **TSS** | Training Stress Score — one workout's load; 100 ≈ 1 h at threshold. |
| **`tss_source`** | Provenance of a TSS value: garmin, manual, power, pace, or hr. |
| **VI** | Variability Index — NP / average power; ride smoothness. |
| **W′ ("W prime")** | Finite work capacity above CP (kJ); the anaerobic battery. |
| **W′bal** | The battery's level over a ride: drains above CP, recharges below. |
| **Zone-referenced target** | A workout step prescribed as a zone, resolved to absolute values from your config. |
